# 极简双端浮动状态栏 - 桌面版独立客户端 (Desktop Standalone Client)

本文件夹包含了为您解耦独立拆分出来的**桌面级可移动、无边框、半透明浮动状态栏小组件**。

此桌面版本不依赖复杂的网页模拟环境，可直接通过 Electron 作为您 PC / Mac 系统底层的超轻型浮动状态显示器。

---

## ✨ 核心特色功能 (Key Enhancements)

1. **真实桌面级悬浮 & 穿透：**
   - 彻底摆脱浏览器 iframe 的隔离阻碍，作为一个拥有真实 OS 级别 `alwaysOnTop`（最高级置顶层级，防遮挡）特性的纯净状态卡。
   - 移除标题栏与窗口框架（Frameless & Transparent Window），完美的磨砂毛玻璃边缘直接无缝挂载于您的真实壁纸之上！

2. **多屏定位与自适应锚定：**
   - 可在控制板中一键切换配置：**极简左下角**、**顶部中央**、**右上角**。
   - 点击切换后，Electron 原生窗口会自动瞬移并贴边对齐该位置，省去手动对齐痛苦！

3. **双重极简收拢机制（自适应悬浮）：**
   - 当检测到鼠标离开状态栏后，自动触发缓冲机制，保留极简 4px 微缩过渡状态或淡入淡出，绝对不干扰正常的写代码与游戏。

4. **无缝拖拽（Webkit Dragging） & 交互避让：**
   - 状态栏的空白区域可直接使用鼠标左键按住并拖动到屏幕任意位置。
   - 功能按钮（如 Codex 锁环、滴水手动重置按钮等）自动避让拖拽（`no-drag` 属性），确保拖拽与交互毫无冲突。

5. **秒级极速联调：**
   - 支持勾选“秒级极速调试模式”，即刻观察到排水从左至右耗尽变红、起跳二次函数曲线发光发热的高保真状态过渡。

## 🚀 极速本地运行指南 (How to Run Locally)

为了在您的 Windows 10/11 或 macOS / Linux 桌面直接启动本系统，请参考以下极简命令：

### 1. 手动提取或下载
在 AI Studio 的右上角设置菜单中，选择 **“Export ZIP”** 或 **“Sync to GitHub”** 将全部工程下载或同步到本地电脑上。

### 2. 进入当前独立目录
打开您的终端（如 PowerShell, cmd, Terminal），并切入解耦后的本独立子文件夹：
```bash
cd desktop-app
```

### 3. 安装依赖（确保本地已安装 Node.js 16+）
```bash
npm install
```

### 4. 唤起独立的小组件
```bash
npm start
```
*此时，您的屏幕左下角（或设定的对齐点）将瞬间被唤醒出一个绝美的、可自由交互的磨砂浮动状态栏！*

## 📦 一键打包成绿色免安装 (.exe / .app) 程序
若您想直接在本机编译生成一键安装的发行包或绿色版，可根据您所在的系统平台，在终端依次键入以下命令：

* **Windows 系统 (一键打包成 Portable 绿色免安装版 .exe):**
  ```bash
  npm run pack:win
  ```
  *(编译后的独立单件 exe 程序将会自动输出在 `dist/` 文件夹内，可直接发送到桌面或发送给好友运行！)*

* **macOS 苹果系统 (打包成 dmg 镜像文件):**
  ```bash
  npm run pack:mac
  ```

* **Linux 标准版:**
  ```bash
  npm run pack:linux
  ```

## 🛠️ 文件构成清单 (Internal Architecture)

- **`main.js`**: Electron 主进程配置文件。设定层级、多端自适应瞬移 API、以及点击展开/收阖设置时极其顺滑的纵向尺寸热拉伸。
- **`preload.js`**: 渲染进程与操作系统主程序的高安全通信网桥。
- **`index.html`**: 极致精简、离线优先的单页面状态组件中心。内置完整的 Canvas 圆环、滴水二次函数起跳动画、和三套原生配色皮肤（磨砂暗卡、毛玻璃白、深夜霓虹）。
